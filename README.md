# vespaPiemonte

Repository archiviato contenente gli script utilizzati per la pubblicazione
dei dati sulla Vespa velutina in Piemonte (Italia).

## Stato del progetto

- Le fonti dati non sono più aggiornate e il sito web non viene più pubblicato.
- I workflow GitHub Actions per la pubblicazione sono stati disattivati.
- Il codice resta disponibile come riferimento per le soluzioni adottate.

## Utilizzo locale

Il progetto può comunque essere ricostruito in locale con [Quarto](https://quarto.org/)
per consultare gli esempi di configurazione e di visualizzazione.
